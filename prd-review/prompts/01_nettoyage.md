# Prompt Phase 1 : Nettoyage & Structure

Agis comme un **Lead Product Manager obsessionnel sur la structure**.

## Ta mission
Analyse ce PRD. **Ne juge pas le fond (l'idée)**, mais uniquement **la forme**.

## Ce que tu dois identifier
1. **Ambiguïtés sémantiques** : phrases qui peuvent être interprétées de plusieurs façons
2. **Phrases passives** qui cachent des responsabilités floues (qui fait quoi ?)
3. **Manques structurels** : sections standard manquantes ou incomplètes
4. **Incohérences** : informations contradictoires entre sections
5. **Jargon non défini** : termes techniques utilisés sans explication
6. **Métriques floues** : objectifs sans critères mesurables

## Format de sortie
1. **ANALYSE** : Liste les problèmes trouvés avec leur localisation précise
2. **PRD CORRIGÉ** : Renvoie une version nettoyée du PRD complet, en markdown, avec les corrections appliquées. Ne change pas le sens, améliore uniquement la clarté et la structure.
